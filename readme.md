# Проект Мейн-Кун

Сайт разработан на HTML, CSS, Javascript. 
Адаптация страницы осуществлена на нескольких контрольных точках:

* 1200px
* 1100px
* 900px
* 600px

## Точу навыки в:
* HTML
* CSS
* JavaScript
* Flexbox-вёрстка
* Позиционирование элементов
* Grid Layout
* Медиазапросы
